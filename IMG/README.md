# Immagini del Sito

Questa cartella contiene tutte le immagini utilizzate dal sito Jekyll.

### Featured Story
- `story.jpg` - The Lifecycle of a Tent

## Come Aggiungere le Immagini

1. Scarica o crea le immagini
2. Salva i file con i nomi esatti sopra indicati
3. Posiziona i file in questa cartella (`IMG/`)
4. Ricostruisci il sito con `bundle exec jekyll build`

## Formato Consigliato

- **Formato**: JPG, PNG o SVG
- **Dimensione prodotti**: 800x1000px (aspect ratio 3:4)
- **Dimensione materiali**: 600x600px (quadrato)
- **Dimensione story**: 1200x675px (aspect ratio 16:9)
